# RankFactory Landing Page

A high-conversion landing page for NEB Class 12 Science exam preparation, integrated with the eTutorClass API.

## 🚀 Features
- **Instant Registration:** Direct integration with Laravel Backend.
- **Dynamic Institutions:** AJAX-powered modal to add new institutions on the fly.
- **Smart Redirect:** Existing users are automatically recognized and sent to the success state.
- **App Deep Linking:** Quick buttons for Play Store, App Store, and WhatsApp Community.
